Eyebrak – Version 1.1
Release Date: 2026-03-12

Overview:
A major update focusing on reliability, modern UI/UX, and enhanced break enforcement.

🚀 Features & Improvements – v1.1
1. Fix Timer Reset Issue:
   - Timer now persists correctly even if the background service worker restarts.
2. Modern Design System:
   - Completely redesigned UI with a premium Emerald & Sage Green palette.
   - Glassmorphism effects and modern typography (Inter).
   - Card-based layout for Settings and Stats.
3. Interactive Visual Timer:
   - Visual phase indicators (Work/Break) in the popup.
4. Hard Break Overlay:
   - New full-screen dimmed overlay on webpages during Pomodoro breaks to enforce resting.
5. Modern In-Page Toasts:
   - Replaced basic system notifications with sleek, interactive in-page toasts.
6. Enhanced Analytics:
   - New visual dashboard with a "Wellness Score" tracking your consistency.

🐛 Bug Fixes – v1.1
- Fixed a critical bug where the timer would reset to its full duration on extension updates or browser restarts.
- Optimized storage usage for better performance.
- Fixed layout issues on the Options page for smaller screens.

📌 Notes
- Version 1.1 marks the transition to a more "Pro" wellness tool with active break enforcement.
- Toasts and Overlays require permissions for all sites to function correctly.
